题解 AT3919 【[ABC088B] Card Game for Two】

2020-03-05 1 min read # 题解

萌新的奇怪思路：

  • nnn mod 2=0:每一次都用奇数项减去偶数项，再加入ansansans。

  • nnn mod 2=1:由于最后一张卡牌肯定是先手的，所以直接把最后一个数加入ansansans，然后n−1n-1n−1。

代码：

#include<bits/stdc++.h>
using namespace std;
int a[101],n,ans;
int cmp(int a,int b)
{
    return a>b;
}
int main()
{
    cin>>n;
    for(int i=1;i<=n;i++) cin>>a[i];
    sort(a+1,a+1+n,cmp);
    if(n%2==1)//直接判断
    {
        n-=1;
        ans+=a[n+1];
    }
    for(int i=1;i<=n;i+=2)//两个一起
    {
        ans+=a[i]-a[i+1];
    }
    cout<<ans;
    //system("pause");
}
